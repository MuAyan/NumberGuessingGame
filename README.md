# NumberGuessingGame

## Overview

This is a simple command-line game made using python.

The player tries to guess a randomly generated number within a given range.

The game provides feedback on how close the guess is to the correct number.

Varying difficulties and subtle hints are added to make the game more engaging.


## Features

- Selectable difficulty levels (1-3)
- Input validation to handle invalid guesses
- Distance-based hints: Far Away, Close, Very Close
- Limited number of guesses each round
- Option to play again






## How To Run:
  1. Clone the repository:
       ```
       git clone https://github.com/MuAyan/NumberGuessingGame.git
       ```
  2. Navigate to the project folder:
       ```
       cd NumberGuessingGame
       ```
  3. Run the game:
       ```
       python NumberGuessingGame.py
       ```
